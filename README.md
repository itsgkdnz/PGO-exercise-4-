# Insurance Office Project

This project is a simple Java console application for managing vehicle insurance policies.

## Features

* Create and store insurance policies
* Calculate final annual premium
* Calculate renewal premium for the next year
* Generate a simple report
* Search policy by number
* Count high-risk policies
* Use of `equals()`, `toString()`, static field and static method

## Classes

* `Policy` → represents one insurance policy
* `InsuranceOffice` → manages a list of policies
* `Main` → runs the example scenario

## Example

The program creates several policies, adds them to the insurance office, and prints calculations and reports in the console.

## Technologies

* Java
* ArrayList
* Object-Oriented Programming
